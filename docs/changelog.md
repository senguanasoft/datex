# Changelog

All notable changes to DateX will be documented in this file.

## [1.0.0] - 2024-01-07

### Added

- Initial release of DateX
- Complete date range picker functionality
- CSS selector support for element targeting
- Multiple built-in themes (Default, Bootstrap, Material)
- Spanish localization with time support
- Time picker with 12/24 hour formats
- Predefined ranges with always-visible calendars
- Comprehensive accessibility support
- TypeScript definitions
- Framework integration examples
- Extensive documentation

### Features

- **Element Selection**: Support for CSS selectors (#id, .class, [attribute])
- **Themes**: Three built-in themes with dynamic theme switching
- **Localization**: Spanish locale with date and time formats
- **Time Picker**: Full time selection with increments and seconds
- **Ranges**: Predefined ranges that don't hide calendars
- **Accessibility**: WCAG 2.1 AA compliant with keyboard navigation
- **Events**: Comprehensive event system for integration
- **Styling**: Exact CSS values matching vanilla-datetimerange-picker

### Technical

- ES modules only (no CommonJS)
- Modern browser support
- Zero dependencies
- TypeScript support
- Comprehensive test coverage
- VitePress documentation site

### Browser Support

- Chrome 88+
- Firefox 85+
- Safari 14+
- Edge 88+
